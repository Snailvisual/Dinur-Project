Kebijakan Privasi Socialflow

Kami menghargai privasi Anda. Kebijakan ini menjelaskan bagaimana Socialflow mengumpulkan, menggunakan, dan melindungi data pribadi Anda.

1. Data yang Dikumpulkan

Socialflow dapat mengumpulkan data seperti nama, email, nomor HP, dan data insight media sosial yang Anda input atau izinkan.

2. Penggunaan Data

Data digunakan untuk keperluan analisis, personalisasi fitur, dan peningkatan layanan. Kami tidak membagikan data ke pihak ketiga tanpa izin Anda.

3. Keamanan Data

Kami berkomitmen menjaga keamanan data Anda dengan standar terbaik. Namun, tidak ada sistem yang 100% aman.

4. Hak Pengguna

Anda berhak mengakses, memperbaiki, atau menghapus data pribadi Anda kapan saja melalui fitur aplikasi.

5. Perubahan Kebijakan

Kebijakan privasi dapat diperbarui sewaktu-waktu. Perubahan akan diinformasikan melalui aplikasi atau email.

6. Kontak

Jika ada pertanyaan terkait privasi, silakan hubungi admin Socialflow.

Terima kasih telah mempercayakan data Anda kepada Socialflow.
